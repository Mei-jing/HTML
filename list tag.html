什么是html

HTML (HyperText Markup Language) 不是一种编程语言

  • 它是一种标记语言，用于告诉您的浏览器如何构造您访问的网页。
  • 它可以像 Web 开发人员希望的那样复杂或简单。
  • HTML 由一系列的 元素组成, 您可以使用它来封装，包装或标记内容的不同部分，使其以某种方式显示，或以某种方式执行。
  • 封闭 tags 可以使一点内容成为超链接以链接到 Web 上的另一页面，斜体字等等。
